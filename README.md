# SpotifyCompressionProject
by Catherina Xu and Shivaal Roy

## Getting Started
To get going, run 
```shell script
bash setup.sh
```
This brew installs Python 3 if not already on your system, creates a Python 3
 virtual environment and then installs necessary packages.
